# 单字母发音差异Diferencias de pronunciación de letras individuales

---

## C Z S

1. “ c ” 在元音 “e和i”前，发[θ]音，即有一个咬舌的动作，发舌尖齿间擦清辅音，发音时舌尖从上下门齿之间微微伸出，气流即可通过，声带不用振动。当然，“c”另外一种情况即为与元音字母 “a，o，u”组成的[k]音。

2. “ z ” 在任何一个元音前都发[θ]音，即任何组合情况下都发与“ce，ci”相同的咬舌音。请注意,当字母“z”与元音“e，i”组合时，同样应书写为“ce，ci”，不存在“ze，zi”的拼写，其两者发音完全一致。

3. “ s ” 任何一个元音前都发[s]音, 即非咬舌清辅音。
- /θ/要把舌尖咬在上牙和下牙之间

- /s/则是牙齿闭合，嘴唇微张，声带不振动，模仿蛇的嘶嘶声就可以了。
