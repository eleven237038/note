# 单元词汇Palabras de las unidades

| 缩写      | 全称               | 中文含义  |
|:------- |:---------------- |:----- |
| m.      | **masculino**    | 阳性    |
| f.      | **femenino**     | 阴性    |
| adj.    | **adjetivo**     | 形容词   |
| adv.    | **adverbio**     | 副词    |
| tr.     | **transitivo**   | 及物动词  |
| intr.   | **intransitivo** | 不及物动词 |
| prnl.   | **pronominal**   | 代词式动词 |
| prep.   | **preposición**  | 介词    |
| conj.   | **conjunción**   | 连词    |
| interj. | **interjección** | 感叹词   |
| pron.   | **pronombre**    | 代词    |
| pl.     | **plural**       | 复数    |

### 重点语法

**1. 命令式 + 人称代词**

- 肯定命令式：代词置于动词之后，连写
  - Compralo, damelo, levantate
- 否定命令式：代词置于动词之前，分写
  - No lo compres, no me lo des, no te levantes

**2. 不定代词**

- alguien / nadie（有人/没人）
- algo / nada（某事/无事）
- ninguno（没有一个）

**3. 副动词（gerundio）**

- 与estar连用表示"正在做..."
- 第一变位（-ar）：-ando
- 第二、三变位（-er, -ir）：-iendo

**4. 条件句**

- si + 现在时 + 将来/命令式
  - Si estudias mucho, aprobars.

### 重点语法

**1. 陈述式现在时**

- 描述当前正在发生的动作或存在的状态
- 表达习惯性或永恒真理

**2. 比较级**

- más + adj/adv + que（比...更...）
- menos + adj/adv + que（比...更不...）
- tan + adj + como（和...一样...）

**3. 最高级**

- el/la/los/las + más + adj（最...的）
- muy + adj（非常...）

**4. 命令式**

- 肯定命令：动词 + 代词（连写）
- 否定命令：no + 代词 + 动词（分写）

**5. 副动词（Gerundio）**

- estar + gerundio 表示正在进行
- -ar → -ando
- -er/-ir → -iendo

**6. 常用句型**

- ¿Qué tiempo hace hoy?（今天天气怎么样？）
- Me gusta / Me interesa（我喜欢/感兴趣）
- Es necesario + inf.（有必要...）
