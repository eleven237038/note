# 单元词汇Palabras de las unidades

| 缩写      | 全称               | 中文含义  |
|:------- |:---------------- |:----- |
| m.      | **masculino**    | 阳性    |
| f.      | **femenino**     | 阴性    |
| adj.    | **adjetivo**     | 形容词   |
| adv.    | **adverbio**     | 副词    |
| tr.     | **transitivo**   | 及物动词  |
| intr.   | **intransitivo** | 不及物动词 |
| prnl.   | **pronominal**   | 代词式动词 |
| prep.   | **preposición**  | 介词    |
| conj.   | **conjunción**   | 连词    |
| interj. | **interjección** | 感叹词   |
| pron.   | **pronombre**    | 代词    |
| pl.     | **plural**       | 复数    |

## Unidad 1 ¿Quiénes somos?

| 单词          | 例句                                                      | 释义                          |
|:----------- |:------------------------------------------------------- |:--------------------------- |
| nombre      | Me llamo Carlos y mi **nombre** es muy común en España. | `m`名字                       |
| país        | China es un **país** muy grande.                        | `m`国家                       |
| año         | Juan tiene veinte **años**.                             | `m`年；岁                      |
| lengua      | El español es una **lengua** muy bonita.                | `f`语言                       |
| estudiante  | Soy **estudiante** en la universidad.                   | `m.f`学生                     |
| médico,a    | Mi hermana es **médica** en un hospital.                | `m.f`医生                     |
| futbolista  | Lionel Messi es un **futbolista** famoso.               | `m.f`足球运动员                  |
| China       | **China** tiene una historia muy larga.                 | 中国                          |
| chino, na   | Mi amigo es **chino** y vive en Pekín.                  | `m.f`中国人；`adj`中国的；`m`汉语     |
| España      | Madrid es la capital de **España**.                     | 西班牙                         |
| español, la | Ella habla muy bien el **español**.                     | `m.f`西班牙人；`adj`西班牙的；`m`西班牙语 |
| Inglaterra  | Londres está en **Inglaterra**.                         | 英国                          |
| inglés, sa  | El profesor de **inglés** es muy simpático.             | `m.f`英国人；`adj`英国的；`m`英语     |
| ser         | Yo quiero **ser** profesor de idiomas.                  | `intr`是                     |
| llamarse    | ¿Cómo puede **llamarse** este lugar?                    | `prnl`叫……名字                 |
| hablar      | Ella puede **hablar** español e inglés.                 | `tr`讲（某种语言）                 |
| tener       | Yo quiero **tener** un coche nuevo.                     | `tr`有                       |
| de          | Soy **de** China y vivo en Pekín.                       | `prep.` 表示来源和所属             |
| y           | Juan **y** María son estudiantes.                       | `conj.` 和                   |
| sí          | **Sí**, quiero estudiar español contigo.                | `adv.` 是，对                  |
| no          | **No**, hoy no tengo tiempo para salir.                 | `adv.` 不是，错                 |
| hola        | **Hola**, ¿cómo estás hoy?                              | `interj.` 你好                |
| cómo        | ¿**Cómo** se dice esto en español?                      | `adv.` 怎样，如何                |
| dónde       | ¿**Dónde** está la biblioteca de la universidad?        | `adv.` 哪⾥                   |
| cuánto      | ¿**Cuánto** cuesta este libro antiguo?                  | `adj.` 多少                   |
| qué         | ¿**Qué** hora es ahora mismo?                           | `pron.,adj.` 什么             |

| amigo, a     | Juan es mi mejor **amigo**.                        | `m.f`朋友             |
| director, ra | La **directora** de la empresa es muy inteligente. | `m.f`经理             |
| italiano, a  | Este queso es **italiano**.                        | `m.f`意大利人；`adj`意大利的 |
| teléfono     | ¿Cuál es tu número de **teléfono**?                | `m`电话               |
| bombero, a   | Mi hermano quiere ser **bombero**.                 | `m.f`消防员            |
| muy          | El café está **muy** caliente.                     | `adv`很              |
| bien         | Todo va muy **bien**, gracias.                     | `adv`好              |
| estar        | Yo quiero **estar** con mis amigos ahora.          | `intr`在；处于（某种状态）    |
